华侨文化寻根的建筑语言重构 —— 以赤坎古镇、立园与仓东村为例 ​

一、引言 ​

1.1 研究背景 ​

近代五邑侨乡处于中西文化剧烈碰撞的前沿。自晚清解除海禁，大批开平籍华侨远赴北美谋生，在排华政策、种族歧视、生存压力下形成强烈的乡土依恋、宗族认同、家国意识。归国后，他们以建筑为载体，将西方技术、形制、审美与岭南传统、儒家伦理、风水信仰、防御需求结合，催生了碉楼、骑楼、侨园、祠堂学校等独特建筑类型。

赤坎古镇作为侨乡商贸与公共文化中心，立园作为旅美华侨谢维立建造的私家园林，二者均由留美归侨群体主导营造，是研究华侨文化寻根与建筑语言重构的典型样本。

1.2 研究对象与范围 ​

赤坎古镇：以骑楼街、碉楼群、宗祠、图书馆、侨墟为核心，代表公共性、商贸性、社群性寻根建筑。

立园：以别墅区、大花园、运河、泮立楼、毓培别墅为核心，代表私属性、家族性、精英性寻根建筑。

1.3 研究意义 ​

揭示华侨生平经历如何转化为建筑形式语言；

厘清文化寻根在建筑上的具体重构表现；

为侨乡遗产保护、文旅活化、地域设计提供理论依据。

二、核心主体：谢氏家族生平经历与文化心理 ​

2.1 谢氏家族三代华侨轨迹 ​

第一代 谢曰佑赴旧金山谋生，创办中药商号，完成从劳工到商人的转变，确立 “海外打拼、回乡兴家” 的家族初心。

第二代 谢圣泮将生意拓展至芝加哥，历经地震、火灾、排华浪潮，深刻体会华人在海外的弱势地位，形成安全至上、家族抱团、光宗耀祖的强烈意愿，是立园建造的发起者。

第三代 谢维立（立园建造者）1893 年生于开平，6 岁赴美，接受完整西方教育，精通商贸与现代生活；1926 年遵父命回乡，耗时十年建成立园。其人生经历塑造了三重身份：

西方现代文明接受者：认同公共空间、轴线规划、舒适居住、现代设备；

岭南宗族文化继承者：坚守祭祖、风水、睦乡、齐家；

排华创伤亲历者：追求体面、防御、尊严，以建筑重建身份认同。

2.2 谢氏驱动的核心建筑逻辑 ​

守根：敬祖、风水、宗族、教化、家国；

实用：防御、交通、通风、排水、生产；

融新：西方材料、结构、柱式、外廊、设备；

开放：公共共享、乡邻共乐、面向现代。

这一逻辑同样主导赤坎古镇关氏、司徒氏等归侨家族的建设行为，使两地建筑呈现同源重构特征。

三、华侨文化寻根的建筑语言重构：赤坎古镇 vs 立园 ​

3.1 空间语言重构：从封闭内向到开放秩序 ​

（1）立园：宅园分区，精英开放 ​

传统岭南园林：宅园合一、紧凑内向、曲径通幽。（见图2）

重构表现：划分为别墅区、大花园区、小花园区，以运河、轴线统筹，居住私密、园林开放。（见图4）

寻根内涵：实现 “欢迎梓里同游” 的宗族理想，把西方公共精神转化为乡土共享空间，是开放型寻根。（见图3、4）

（2）赤坎古镇：梳式巷弄 → 骑楼大街 ​

传统村落：封闭梳式布局、窄巷、内向天井。

重构表现：规整街道、连续骑廊、对称立面、公共广场。（见图5、6）

寻根内涵：以西方城市形态重建侨乡体面与公共秩序，是社群型寻根。

3.2 功能语言重构：从单一使用到复合共生 ​

（1）立园：居住 — 观赏 — 防御 — 生产 — 祭祀复合 ​

运河承担运输、防洪、灌溉、逃生；（见图7）

建筑底层会客、中层居住、顶层祭祖；（见图10、图11、图12）

花园兼具观赏、种植、社交、教化。（见图8、图9）

寻根内涵：建筑服务于家族生存、文化传承、家国理想。

（2）赤坎古镇：商贸 — 防御 — 教育 — 祭祀复合 ​

骑楼下商上宅；

碉楼战时防御、平时储物 / 居住；

宗祠前祠后校，祭祀与办学合一。

寻根内涵：以复合功能维系乡土经济、宗族秩序、文化延续。

3.3 造型语言重构：中式为体，西式为用 ​

（1）立园：顶级中式屋顶 + 西方古典立面 ​

庑殿顶、绿色琉璃瓦、龙脊吻兽（中式最高等级）；（见图13）

爱奥尼柱、科林斯柱、帕拉迪奥母题、三段式立面（西方古典）。（见图14）

重构逻辑：精神归宗，形式现代。

（2）赤坎：骑楼 / 碉楼西式表皮 + 中式内核 ​

拱券、山花、柱式、阳台；

内部保留天井、厅堂、神位、廊道。（见图15）

重构逻辑：外表现代化，内核乡土化。（见图16）

3.4 结构与材料重构：传统土木 → 钢筋混凝土防御体系 ​

（1）立园 ​

钢筋混凝土框架、德国钢、英国水泥、意大利水磨石；厚墙、铁窗、多层防护。来源：家族安全需求 + 西方工业技术。（见图17）

（2）赤坎碉楼 ​

混凝土墙体、射击孔、燕子窝、铁闸门、厚重石门。来源：侨乡匪患 + 华侨联防需求。共同寻根意义：以最强技术守护最根本的家。（见图18）

3.5 装饰语言重构：文化认同的视觉表达 ​

（1）立园：儒家教化 + 家国图强 ​

灰塑题材：三顾茅庐、张九龄点兵、求贤救国；楹联核心：立身为齐家之本，园林乃救国之基。寻根指向：以建筑育人，坚守中华文化正统。（见图19、图21）

（2）赤坎：新生活意象 + 民权进步 ​

壁画：轮船、火车、洋房、汽车；楹联：共和、民权、振兴中华、世界文明。寻根指向：以现代符号寄托家乡富强、民族自尊。（见图20）

3.6 风水与信仰重构：传统内核，现代形式 ​

立园设德国钢打虎鞭镇虎山，以西物行中式风水；（见图23）

仓东村背山面水、门楼朝向、宗祠居中，风水秩序不变。（见图24、图25）

寻根本质：信仰不丢，工具更新。

四、文化地域性对于设计与利用的参考价值建议 ​

4.1 文化地域性核心特征提炼 ​

生存地域性：炎热多雨、洪涝匪患，决定通风、遮阳、排水、防御优先。

宗族地域性：聚族而居、敬祖重教，决定宗祠、祖先堂、教化装饰。

侨民地域性：跨洋经历、中西双重文化，决定折衷风格、复合功能。

务实地域性：重功用、轻礼制、强适应，决定材料就地 / 进口结合、形式服务生活。

4.2 对遗产保护的参考建议 ​

坚持 “真实性保护”优先保护结构、空间、材料、装饰四大核心要素，不随意仿古翻新，保留历史痕迹。

坚持 “整体性保护”赤坎保护骑楼 — 碉楼 — 河道 — 宗祠系统；立园保护别墅 — 运河 — 花园 — 山体格局，不割裂空间关系。

保护 “叙事性”保留华侨生平、家族故事、楹联题刻、风水布局，让建筑可解读、可共情。

4.3 对文旅活化的参考建议 ​

空间活化遵循原功能逻辑赤坎骑楼恢复商业 + 文创 + 展览；立园恢复居住体验 + 园林游赏 + 文化展示。

场景化讲述华侨寻根故事以谢氏家族为主线，呈现 “出洋 — 谋生 — 归乡 — 建园” 完整叙事，增强情感共鸣。

适度现代化，不破坏地域性配套设施采用现代材料、传统形制，保持风貌统一。

4.4 对当代地域建筑设计的参考建议 ​

功能优先，回应地域气候与生活延续骑楼外廊、立园通风屋顶、天井冷巷等低技生态策略。

形式简约转译，不符号堆砌提取柱式比例、屋顶轮廓、对称轴线、装饰母题，现代重构。

文化在地，植入地域精神强调归属感、家族性、公共性、务实性，让建筑扎根地方文化。

材料本土与现代结合延续混凝土、砖、石、钢、木的混合使用传统，强调坚固耐久与地域质感。

4.5 对文化传承与教育的参考建议 ​

建立侨乡建筑文化课程体系以赤坎、立园为课堂，讲解华侨史、建筑语言、中西交流。

推动 “可参与式” 传承鼓励灰塑、砖雕、壁画等工艺传承，让传统技艺回归当代生活。

五、结论 ​

赤坎古镇与立园的建筑语言重构，是近代华侨文化寻根最直观的物质呈现。以谢氏家族为代表的归侨群体，在海外漂泊受辱、归乡重建尊严的过程中，将乡土之根、宗族之根、文化之根、家国之根注入建筑，形成一套独特的侨乡建筑体系。

立园以私家园林完成精英阶层的文化回归，追求家族体面与精神安顿；赤坎古镇以公共侨墟完成社群层面的文化重建，追求地方秩序与文明进步。二者虽形态不同，却共享同一套文化逻辑：以中式为魂、以西式为形；以传统为根、以现代为用；以生存为基、以尊严为目标。

文化地域性为当代保护与设计提供了清晰方向：只有尊重地域气候、生活、历史、情感，才能让建筑真正扎根土地、延续文脉。赤坎与立园的价值，不仅在于中西合璧的形式美感，更在于它们记录了一代华侨身在海外、心向故土、魂系中华的最深层寻根之路。
