Word转Markdown在线工具 ​

功能 ​

  • 上传Word文档，自动提取正文内容
  • 上传图片文件夹，自动按1.jpg、2.jpg、3.jpg...编号
  • 一键生成Markdown文件和相关文件夹下载

使用方法 ​

  1. 打开 index.html 即可使用（无需服务器）

技术说明 ​

  • 使用 mammoth.js 解析 Word 文档
  • 使用 JSZip 生成下载文件
